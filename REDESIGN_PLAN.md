## Целевой дизайн-язык (из референса)
- Светлая основа: фон страницы #f4f5f7, карточки белые, радиус 20–24, мягкие тени.
- Тёмные акцент-блоки: карта, тикер, featured-элементы — глубокий графит #0b0d12.
- Шапка: светлая, логотип «Logi**Core**» (Core — синий #1677ff), навигация — ПИЛЮЛИ
  (активный пункт — чёрная пилюля с белым текстом), справа поиск/уведомления/тема/профиль.
- Тикер живых событий под шапкой: бегущая строка «LC-XXXX · событие» с цветными точками статусов.
- Хиро секций: огромный заголовок слева (34–44px, weight 800), крупные метрики справа
  (число + подпись + цветной суб-лейбл: «+5,4%», «по графику», «внимание ⚠»).
- Action-bar: белая пилюля-строка (контекст слева, предупреждения + действия справа,
  главная кнопка — синяя пилюля).
- Featured-карточка заявки: слева таймлайн рейса (погрузка → текущий статус с % → выгрузка,
  итоги: маршрут/стоимость), справа тёмная карта живого отслеживания + строка водителя
  (аватар-инициалы, госномер, кнопки чат/звонок).
- Таблица: табы-фильтры пилюлями со счётчиками, строки с прогресс-баром маршрута
  (цвет = статус: синий в пути / зелёный завершён / красный проблема), водитель с
  аватаром-инициалами, сумма справа жирная tabular-nums.
- Статус-пилюли: уже реализованы (components/ui/StatusPill.tsx) — переиспользовать.

## Стратегия «не сломать»
- Ant Design НЕ выпиливаем разом: он остаётся движком форм/модалок/таблиц.
  Визуально перекрываем собственным скином (класс lc-page + новые lc-* классы в
  apps/web/src/app/globals.css). Отказ от antd — постепенный, экран за экраном.
- Все текущие колонки таблиц сохраняются (референс — неполный, у нас данных больше:
  Организация, Заказчик, Перевозчик, Менеджер, Дата погрузки и т.д.).
- Роль-зависимые данные не светить: маржу показывать только там, где она уже показывается
  (реестр), НЕ добавлять в заявки без проверки роли.
- data-guide-атрибуты не удалять (ИИ-гид). popupClassName у меню навигации не терять.
