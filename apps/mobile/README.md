# LogiCore Driver — мобильное приложение водителя

Expo (React Native) приложение для водителей: активный рейс, статусы,
карта с маршрутом, фото документов, GPS-трекинг в фоне.

## Что умеет

- **Вход по телефону и паролю.** Пароль водителю задаёт компания в карточке
  водителя (Кабинет → Компания → Сотрудники → водитель → «Пароль для
  мобильного приложения»). Эндпоинт: `POST /auth/driver-login`.
- **Экран «Рейс»**: номер, статус-пилюля с прогресс-баром, таймлайн маршрута
  (нумерованные точки, навигатор, звонок контакту точки), груз, кнопка
  смены статуса, «Фото документа» (камера/галерея → ТТН к заявке),
  «Проблема» (сигнал диспетчеру).
- **Экран «Карта»** (Mapbox): маршрут по дорогам (Directions API, фолбэк —
  прямые линии), фирменные нумерованные маркеры (зелёные — погрузка,
  красные — выгрузка), 3D-модель грузовика, светлая/тёмная тема.
- **Экран «Рейсы»**: активные + история (последние 50).
- **Профиль**: фото профиля (загружается на платформу, видно в схеме
  компании), транспорт, настройки, выход.
- **GPS-трекинг**: автоматически включается при активном рейсе, шлёт
  координаты в фоне (`/tracking/gps`); диагностика — Настройки →
  «Проверить GPS».

## Перед сборкой — ОБЯЗАТЕЛЬНО настроить

1. **Адрес API** — `app.json` → `expo.extra.apiUrl`.
   Сейчас стоит адрес для локальной отладки. Для прода укажи публичный
   URL API на Railway, например:
   ```json
   "extra": { "apiUrl": "https://ВАШ-API-ДОМЕН.up.railway.app" }
   ```
2. **Mapbox download token** — `app.json` → плагин `@rnmapbox/maps` →
   `downloadToken`. Сейчас заглушка `YOUR_MAPBOX_DOWNLOAD_TOKEN`.
   Получить: account.mapbox.com → Access tokens → создать **secret token
   (sk.…)** с галочкой `DOWNLOADS:READ`. Без него нативная сборка не
   соберётся. Публичный токен карты (pk.…) уже прописан в
   `app/(tabs)/map.tsx` — при желании замени на свой.

## Запуск в разработке

```bash
cd apps/mobile
npm install            # или pnpm install из корня
npx expo run:android   # нужен dev-клиент (Mapbox не работает в Expo Go)
```
Для локального API на телефоне через USB: `adb reverse tcp:3001 tcp:3001`
и `apiUrl: "http://localhost:3001"`.

## Сборка для магазинов (EAS)

```bash
npm i -g eas-cli
eas login
eas build -p android --profile production   # APK/AAB
eas build -p ios --profile production        # нужен Apple Developer аккаунт
```

## Как выдать водителю доступ

1. В кабинете компании: Компания → Сотрудники → водитель (или создать).
2. Заполнить телефон и «Пароль для мобильного приложения» (мин. 6 символов).
3. Водитель входит в приложение: телефон + этот пароль.
4. Один аккаунт = одно устройство (Single Session Policy): вход на втором
   телефоне разлогинит первый.
